Some Web Design Books Are Dangerous
  • Mar 29, 2005
  • ·
  • css
  • webdev
I recently picked up a copy of the "for Dummies" CSS tutorial book and was disappointed to find the following in a section about using named vs. hex color values (like cornflowerblue instead of #6495ED):
"... my advice is to just assume that pretty much everyone who'll see your web page uses IE [Internet Explorer]. Why? Because most everyone does use IE." p. 79 CSS Web Design For Dummies, 2005
Thankfully, our dear author is misinformed, or simply addled. Unfortunately, the sentiment that "only Internet Explorer matters" is a popular one still. New webmasters please be 'ware: there are alternatives to Microsoft IE, most notably Firefox which has more than 25 million downloads and is now reported by the W3C to have nearly 20% usage on the internet. That's a lot of people who will be seeing a screwed up page if you only test your work in IE. Apparently, people want to test and develop in IE because it simplifies life. "Your job is much easier," writes the www.positioniseverything.net.
