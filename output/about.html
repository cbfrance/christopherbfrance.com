About This Site

nanoc3 is a nanoc3 blog starter kit for Ruby hackers.

The blog compiles into pure HTML. No javascript (keeping it simple). Ultra-fast and light. No database. No headaches. Host anywhere. Source is available from this nanoc3_blog github repository.

Source is provided under the FreeBSD License. All posts are CC-BY Chris Blow.
